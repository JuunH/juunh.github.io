About

My name is Nicholas. I am currently a college student aiming to enroll into Newcastle University to study Cyber Security / Software Engineering.

I am primarily focusing on building my knowledge foundation by exploring various IT and Computer Science fields to prepare myself for university and my future career.

Currently, I am learning front-end Web Development, marking up my own website with HTML, SCSS and Javascript. I have this site hosted on Github to learn about Github's workflow, version control and developer collaboration while at the same time, I have set up my own local web server using a Raspberry Pi to use as my own sandbox environment. This will help me learn PHP, MySQL and the back-end fundamentals of websites.

Learning these skills will broaden my general understanding of how different systems function and how I can set-up my own to expand upon it.
